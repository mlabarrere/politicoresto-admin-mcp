# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cov] as its code
of conduct. The full text — including examples of acceptable and unacceptable
behaviour, enforcement responsibilities, and the escalation ladder — lives at
the canonical URL and is incorporated here by reference.

[cov]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

The code of conduct applies in every project space — issues, pull requests,
discussions, commit messages, code comments — and in every public space where
someone is representing the project (conference talks, social-media posts, and
so on).

## Reporting

Abusive, harassing, or otherwise unacceptable behavior may be reported to the
maintainer via the address listed in [SECURITY.md](SECURITY.md) as the
`SECURITY_CONTACT`. All reports are handled confidentially.

If you do not receive an acknowledgement within five business days, please
escalate by opening a minimal issue asking the maintainer to check their
inbox (do not include the subject of the report in public).

## Enforcement

The maintainer will follow the four-step
[Community Impact Guidelines][guidelines] from the Contributor Covenant when
determining the consequences of a violation.

[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
